# Slack Bot を作る

# 技術選定

- TypeScript
- Slack Bolt
- ソケットモードでボットを動作させる

# 仕様

- 起動時にボタン付きメッセージ送信する。
- そのボタンを押すと、ボタンを押したユーザーを "#2026-" から始まる全チャンネルへ自動で招待する ("#2026-" という名前は後に変えるかもしれないので、定数にしておく)。
- これの達成のためには、チャンネル一覧をあらかじめ取得しておく必要があるかも。
- エラーが起きてもボットが止まらないようにする。
- 特定の指定チャンネルに入ったときに "#2026-" から始まる全チャンネルへ自動へ招待する。
